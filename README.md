# Maniac Forum UI

## Hilfe

### Kontext-Steuerung

    <Tab> - Fokus-Wechsel auf Boards, Threads, Unterthreads und Nachrichten
     ↓ ↑  - Zur Auswahl im aktuellen ausgewählten Menü
     ← →  - Auswahl des Boards
       ?  - Hilfsseite

### Globale Steuerung

       j  - Nächster Unterthread
       k  - Vorheriger Unterthread
       J  - Nächster Thread
       K  - Vorheriger Thread
